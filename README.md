# Employees CLI

Консольное приложение для работы со **справочником сотрудников** на **PostgreSQL**: создание схемы/таблицы, добавление записей, выборки, массовая генерация и замер времени.

## Установка и настройка

Требуются **Node.js 18+** и **PostgreSQL**. Создайте файл **.env** с переменными: `PGUSER`, `PGHOST`, `PGDATABASE`, `PGPASSWORD`, `PGPORT`. Затем установите зависимости и соберите проект:

```bash
npm i
npm run build
```

## Запуск режимов

Запускайте так: `npm run start <режим>` или `node dist/app.js <режим>`.

```bash
# 1. Создать схему app и таблицу app.employees
npm run start 1

# 2. Добавить сотрудника (ФИО можно с/без кавычек), дата YYYY-MM-DD, пол Male|Female
npm run start 2 "Ivanov Petr Sergeevich" 2009-07-12 Male

# 3. Вывести строки с уникальным ФИО+датой, отсортированные по ФИО, + возраст
npm run start 3

# 4. Сгенерировать 1 000 000 записей (+100 с Male и фамилией на F), вставка пакетами
npm run start 4

# 5. Выборка: Male и last_name LIKE 'F%', показать время выполнения
npm run start 5
```

## Примечания

Имена вводятся **на английском**, дата строго **YYYY-MM-DD**. Словари имён копируются в `dist` при `npm run build`; пути к ассетам резолвятся относительно модуля (через `import.meta.url`).
